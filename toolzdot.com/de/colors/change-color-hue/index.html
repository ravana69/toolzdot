Was ist Farbton?

Farbton ist das Attribut der Farbe, das durch seine dominante Wellenlänge bestimmt wird. Der Farbton lässt eine Farbe „rot“, „grün“ oder „blau“ aussehen. Mit anderen Worten, der Farbton ist die reinste Form einer Farbe. Die Farbtöne sichtbarer Lichtwellen reichen von Rot bis Violett, mit Orange, Gelb, Grün, Blau und Indigo dazwischen.

Der Farbton einer Farbe kann geändert werden, indem Wellenlängen zur dominanten Wellenlänge der Farbe addiert oder subtrahiert werden. Wenn Sie beispielsweise rotes Licht zu grünem Licht hinzufügen, ist die resultierende Farbe gelb. Dies liegt daran, dass sich Rot und Grün an entgegengesetzten Enden des sichtbaren Lichtspektrums befinden und wenn sie kombiniert werden, erzeugen sie Lichtwellen, die näher am gelben Ende des Spektrums liegen.

Der Farbton kann auch geändert werden, indem einer Farbe weißes oder schwarzes Licht hinzugefügt wird. Das Hinzufügen von weißem Licht zu einer Farbe lässt die Farbe heller erscheinen, während das Hinzufügen von schwarzem Licht die Farbe dunkler erscheinen lässt.

Schließlich kann der Farbton dadurch beeinflusst werden, wie eine Farbe mit anderen Farben gemischt wird. Wenn Sie beispielsweise blaue und gelbe Farbe miteinander mischen, ist die resultierende Farbe grün. Dies liegt daran, dass Blau und Gelb beide Primärfarben sind und wenn sie miteinander gemischt werden, eine Sekundärfarbe erzeugen.

Wie ändere ich den Farbton?

Der Farbgenerator zum Abdunkeln funktioniert, indem er eine bestimmte Farbe nimmt und sie dann um einen bestimmten Betrag abdunkelt. Der Grad, um den die Farbe abgedunkelt wird, kann vom Benutzer gesteuert werden.

  1. Wählen Sie die Farbe aus der Farbauswahl aus oder wenn Sie den Farbwert kennen, können Sie direkt den HEX-, RGB-, HSL- oder HSV-Wert eingeben.
  2. Sie können den Betrag auch anpassen, indem Sie den Schieberegler bewegen.
  3. Sie sehen unten die geänderte Farbtonfarbe Ihrer angegebenen Farbe.
  4. Sie können den Farbwert je nach Bedarf in das RGB-, HEX-, HSV- oder HSL-Format kopieren.
